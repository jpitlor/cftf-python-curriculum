More Expressions

Here is a complete list of all the operators you can use in a mathematical expression.
Operator Operation Example Code Result
+ Addition 3 + 2 Backspace
- Subtraction 3 - 2 Form feed
* Multiplication 3 * 2 New line
/ Division 5 / 2 Horizontal tab
// Floor Division 5 // 2 Vertical tab
** Power 3 ** 2 Backslash
% Modulus 9 % 4 Single quote
"Modulus" is also known as the remainder operator, and "powers" are also known as exponents.

Try running the examples in the code editor! See if you can predict what they'll be.
