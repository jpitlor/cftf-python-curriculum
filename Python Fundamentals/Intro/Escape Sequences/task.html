Escape Sequences

There are lots of special characters that you can print using escape sequences.
Sequence Meaning
\b Backspace
\f Form feed
\n New line
\t Horizontal tab
\v Vertical tab
\\ Backslash
\' Single quote
\" Double quote
\? Question mark

Try running the example in the code editor, and you'll get an error explaining why you need an escape sequence. Then, use the table above to fix it.
