PEMDAS (Mathematical Order of Operations)

In Python, mathematical expressions are evaluated in the following order:

  1. Parentheses
  2. Exponents
  3. Multiplication/Division
  4. Addition/Subtraction

Challenge

Here are three problems. Use Python to solve them!

  • 100 - 4*(7 - 4)^3
  • 20 + 3*(5 - 1)
  • 3 + 2^2*(1 + 8)
Having trouble with those exponents? In Python, you can use the ** operator to signify an exponent.
